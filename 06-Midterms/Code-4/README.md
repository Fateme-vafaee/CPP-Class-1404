# 06-Midterms

## English

### Overview
This folder contains C++ midterm practice exercises.

### Files Description

#### `01_practice.cpp`
- **Purpose**: Prints the Fibonacci series from the first term up to the n-th term.
- **Concepts**: Loops, sequences, arithmetic operations
- **Input**: An integer n
- **Output**: Fibonacci series from the first term to the n-th term

#### `02_practice.cpp`
- **Purpose**: Prints all prime numbers smaller than N.
- **Concepts**: Loops, conditional statements, prime number checking
- **Input**: An integer N
- **Output**: All prime numbers smaller than N

#### `03_practice.cpp`
- **Purpose**: Receives an array of 10 integers from the user and prints its unique elements.
- **Concepts**: Arrays, loops, conditional statements
- **Input**: 10 integers entered by the user
- **Output**: Unique elements of the array

#### `04_practice.cpp`
- **Purpose**: Finds and prints the smallest and largest numbers in an array.
- **Concepts**: Arrays, loops, comparison operations
- **Input**: An array of integers entered by the user
- **Output**: The minimum and maximum values in the array

#### `05_practice.cpp`
- **Purpose**: Receives the scores of 20 students and increases all scores equally so that the highest score becomes 20.
- **Concepts**: Arrays, loops, arithmetic operations
- **Input**: Scores of 20 students
- **Output**: Adjusted scores where the maximum score is 20


## فارسی

### نمای کلی
این پوشه شامل تمرین‌های میان‌ترم C++ است.

### توضیح فایل‌ها

#### `01_practice.cpp`
- **هدف**:جملات سری فیبوناچی را از جمله اول تا جمله n‌ام چاپ می‌کند.
- **مفاهیم**: حلقه‌ها، دنباله‌ها، عملیات ریاضی
- **ورودی**: یک عدد صحیح n
- **خروجی**: سری فیبوناچی از جمله اول تا جمله n‌ام

#### `02_practice.cpp`
- **هدف**: تمام اعداد اول کوچکتر از N را چاپ می‌کند.
- **مفاهیم**: حلقه‌ها، شرط‌ها، بررسی عدد اول
- **ورودی**: یک عدد صحیح N
- **خروجی**: تمام اعداد اول کوچکتر از N

#### `03_practice.cpp`
- **هدف**: یک آرایه ۱۰تایی از کاربر دریافت می‌کند و اعداد غیرتکراری آن را چاپ می‌کند.
- **مفاهیم**: آرایه‌ها، حلقه‌ها، شرط‌ها
- **ورودی**: ۱۰ عدد صحیح که توسط کاربر وارد می‌شوند
- **خروجی**: اعداد غیرتکراری آرایه

#### `04_practice.cpp`
- **هدف**: کوچک‌ترین و بزرگ‌ترین عدد موجود در آرایه را پیدا کرده و چاپ می‌کند.
- **مفاهیم**: آرایه‌ها، حلقه‌ها، عملگرهای مقایسه
- **ورودی**: یک آرایه از اعداد صحیح که توسط کاربر وارد می‌شود
- **خروجی**: کوچک‌ترین و بزرگ‌ترین مقدار موجود در آرایه

#### `05_practice.cpp`
- **هدف**: نمرات ۲۰ دانشجو را دریافت می‌کند و به همه نمرات به‌صورت مساوی مقداری اضافه می‌کند تا بالاترین نمره برابر با ۲۰ شود.
- **مفاهیم**: آرایه‌ها، حلقه‌ها، عملیات ریاضی
- **ورودی**: نمرات ۲۰ دانشجو که از ورودی دریافت می‌شود
- **خروجی**: نمرات اصلاح‌شده به‌طوری که بیشترین نمره برابر با ۲۰ باشد
