# 06-Midterms

## English

### Overview
This folder contains C++ midterm practice exercises.

### Files Description

#### `01_practice.cpp`
- **Purpose**: Reads numbers from the user until -1 is entered, then calculates and prints the average.
- **Concepts**: Loops, conditionals, input/output, arithmetic operations
- **Input**: A series of integers entered by the user (ending with -1)
- **Output**: Average of the entered numbers or a message if no numbers were entered

#### `02_practice.cpp`
- **Purpose**: Prints a right-angled triangle pattern using stars based on the given value of N.
- **Concepts**: Loops, nested loops, output formatting
- **Input**: An integer N
- **Output**: A star pattern where each line contains one more star than the previous line

#### `03_practice.cpp`
- **Purpose**: Reads an array of 10 integers from the user and prints the unique numbers.
- **Concepts**: Arrays, loops, conditional statements
- **Input**: 10 integers entered by the user
- **Output**: The unique numbers from the array

#### `04_practice.cpp`
- **Purpose**: Receives an integer amount from the user and determines the minimum number of coins (1, 2, 10) needed to make that amount.
- **Concepts**: Greedy algorithm, loops, arithmetic operations
- **Input**: An integer representing the amount of money
- **Output**: Minimum number of coins needed to make the amount

#### `05_practice.cpp`
- **Purpose**: Receives an array of 100 numbers from the user and prints the first quartile of the array.
- **Concepts**: Arrays, loops, sorting, basic statistics
- **Input**: 100 numbers entered by the user
- **Output**: Numbers belonging to the first quartile of the array



## فارسی

### نمای کلی
این پوشه شامل تمرین‌های میان‌ترم C++ است.

### توضیح فایل‌ها

#### `01_practice.cpp`
- **هدف**: اعداد را از کاربر دریافت می‌کند تا زمانی که -1 وارد شود و سپس میانگین آن‌ها را محاسبه و چاپ می‌کند.
- **مفاهیم**: حلقه‌ها، شرط‌ها، ورودی/خروجی، عملیات ریاضی
- **ورودی**: مجموعه‌ای از اعداد صحیح که توسط کاربر وارد می‌شوند (-1 برای پایان)
- **خروجی**: میانگین اعداد وارد شده یا پیغام در صورتی که هیچ عددی وارد نشده باشد

#### `02_practice.cpp`
- **هدف**: چاپ یک الگوی مثلثی قائم با استفاده از ستاره‌ها بر اساس مقدار N.
- **مفاهیم**: حلقه‌ها، حلقه‌های تو در تو، قالب‌بندی خروجی
- **ورودی**: یک عدد صحیح N
- **خروجی**: الگوی ستاره‌ای که در هر خط یک ستاره بیشتر از خط قبل چاپ می‌شود

#### `03_practice.cpp`
- **هدف**: یک آرایه ۱۰تایی از کاربر دریافت می‌کند و اعداد غیرتکراری آن را چاپ می‌کند.
- **مفاهیم**: آرایه‌ها، حلقه‌ها، شرط‌ها
- **ورودی**: ۱۰ عدد صحیح که توسط کاربر وارد می‌شوند
- **خروجی**: اعداد غیرتکراری آرایه

#### `04_practice.cpp`
- **هدف**: یک عدد صحیح از کاربر دریافت می‌کند و مشخص می‌کند برای رسیدن به آن مبلغ با سکه‌های ۱، ۲ و ۱۰ تومانی به حداقل چند سکه نیاز است.
- **مفاهیم**: الگوریتم حریصانه، حلقه‌ها، عملیات ریاضی
- **ورودی**: یک عدد صحیح که نمایانگر مقدار پول است
- **خروجی**: حداقل تعداد سکه‌های لازم برای رسیدن به آن مبلغ

#### `05_practice.cpp`
- **هدف**: یک آرایه ۱۰۰تایی از کاربر دریافت می‌کند و اعداد چارک اول آرایه را چاپ می‌کند.
- **مفاهیم**: آرایه‌ها، حلقه‌ها، مرتب‌سازی، آمار پایه
- **ورودی**: ۱۰۰ عدد که توسط کاربر وارد می‌شوند
- **خروجی**: اعدادی که متعلق به چارک اول آرایه هستند
