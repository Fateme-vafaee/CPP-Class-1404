##  How to Compile and Run
If you are new to programming or don't know how to run these files, follow the instructions below.

###  Prerequisites
To run the source codes, you need a C++ compiler installed on your system:
* **Windows:** [MinGW](https://www.mingw-w-pro.org/) or Visual Studio
* **Linux:** `g++` (GCC)
* **macOS:** Xcode Command Line Tools (`clang`)

###  Running via Terminal (Recommended)
1. Open your terminal or command prompt.
2. Navigate to the exercise directory.
3. Use the following commands:

```bash
# Compile the source code
g++ main.cpp -o solution
```
 Run the compiled file
 On **Windows**:
`solution.exe`

 On **Linux/macOS**:
`./solution`
