# 02-Functions

## English

### Overview
This directory contains C++ examples and exercises demonstrating the fundamental concepts of functions, including declaration, parameters, return values, and recursion.

### Files Description

#### `function_example.cpp`
- **Purpose**: Demonstrates basic function declaration, definition, and usage patterns
- **Concepts**: Function syntax, function calls, control flow
- **Input**: Function parameters and user input
- **Output**: Function return values and console output

#### `function_input.cpp`
- **Purpose**: Shows how to pass arguments to functions using different parameter techniques
- **Concepts**: Function parameters, pass-by-value, pass-by-reference, argument passing
- **Input**: Parameters passed to functions
- **Output**: Results based on function logic and parameter handling

#### `function_return.cpp`
- **Purpose**: Demonstrates various return types and return value handling in functions
- **Concepts**: Return types (void, int, double, etc.), return statements, value returning
- **Input**: Function parameters
- **Output**: Returned values from functions

#### `recursive_functions.cpp`
- **Purpose**: Shows recursive function implementation and use cases with examples
- **Concepts**: Recursion, base case, recursive case, call stack, factorial/fibonacci patterns
- **Input**: Parameters for recursive calculations
- **Output**: Results of recursive function execution


## فارسی

### نمای کلی
این پوشه شامل نمونه‌های C++ و تمرین‌هایی است که مفاهیم اساسی توابع شامل اعلان، پارامترها، مقادیر بازگشتی و بازگشت را نشان می‌دهد.

### توضیح فایل‌ها

#### `function_example.cpp`
- **هدف**: اعلان، تعریف و الگوهای استفاده اساسی توابع را نشان می‌دهد
- **مفاهیم**: نحو توابع، فراخوانی توابع، جریان کنترل
- **ورودی**: پارامترهای تابع و ورودی کاربر
- **خروجی**: مقادیر بازگشتی و خروجی کنسول

#### `function_input.cpp`
- **هدف**: نحوه انتقال آرگومان‌ها به توابع با تکنیک‌های مختلف پارامتر را نشان می‌دهد
- **مفاهیم**: پارامترهای تابع، pass-by-value، pass-by-reference، انتقال آرگومان
- **ورودی**: پارامترهای انتقال‌یافته به توابع
- **خروجی**: نتایج بر اساس منطق تابع و مدیریت پارامتر

#### `function_return.cpp`
- **هدف**: انواع بازگشتی مختلف و مدیریت مقادیر بازگشتی از توابع را نشان می‌دهد
- **مفاهیم**: انواع بازگشتی (void، int، double و غیره)، دستورات return، بازگشت مقدار
- **ورودی**: پارامترهای تابع
- **خروجی**: مقادیر بازگشتی از توابع

#### `recursive_functions.cpp`
- **هدف**: پیاده‌سازی توابع بازگشتی و موارد استفاده آن را با نمونه‌ها نشان می‌دهد
- **مفاهیم**: بازگشت، base case، recursive case، call stack، الگوهای فاکتوریل/فیبوناچی
- **ورودی**: پارامترهای محاسبات بازگشتی
- **خروجی**: نتایج اجرای توابع بازگشتی
