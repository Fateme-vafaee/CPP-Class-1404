# 04-Strings 

## English

### Overview
This folder contains C++ programming exercises focused on string manipulation concepts such as searching, reversing, palindrome checking, character counting, and string concatenation.

### Files Description

#### `combine_strings.cpp`
- **Purpose**: Combination of Two Strings Together
- **Concepts**: Strings, Combination
- **Input**: Two strings entered by the user
- **Output**: The combined string

#### `counting_specific_characters.cpp`
- **Purpose**: Counts how many times a specific character appears in a string
- **Concepts**: Strings, loops, character comparison
- **Input**: A string and a character entered by the user
- **Output**: Number of occurrences of the character in the string

#### `palindrome.cpp`
- **Purpose**: Checks whether a string is a palindrome
- **Concepts**: Strings, comparison, loops
- **Input**: A string entered by the user
- **Output**: Indicates whether the string is a palindrome or not

#### `reverse_string.cpp`
- **Purpose**: Reverses a given string
- **Concepts**: Strings, loops, indexing
- **Input**: A string entered by the user
- **Output**: The reversed string (from last character to first)

#### `search_string_in_text.cpp`
- **Purpose**: Searches for one string inside another string and counts how many times it appears
- **Concepts**: Strings, loops, substring search
- **Input**: Two strings entered by the user
- **Output**: Whether the string exists and the number of repetitions



## فارسی

### نمای کلی
این پوشه شامل تمرین‌های C++ با تمرکز بر کار با رشته‌ها (String) است؛ از جمله جستجو در رشته، برعکس‌کردن رشته، بررسی پالیندروم بودن، شمارش کاراکتر خاص و الحاق رشته‌ها.

### توضیح فایل‌ها

#### `combine_strings.cpp`
- **هدف**: الحاق (چسباندن) دو رشته به یکدیگر
- **مفاهیم**: رشته‌ها، الحاق رشته‌ها
- **ورودی**: دو رشته از کاربر
- **خروجی**: رشته‌ی نهایی ترکیب‌شده

#### `counting_specific_characters.cpp`
- **هدف**: شمارش تعداد تکرار یک کاراکتر خاص در یک رشته
- **مفاهیم**: رشته‌ها، حلقه‌ها، مقایسه کاراکتر
- **ورودی**: یک رشته و یک کاراکتر از کاربر
- **خروجی**: تعداد دفعات تکرار کاراکتر در رشته

#### `palindrome.cpp`
- **هدف**: بررسی پالیندروم بودن یک رشته
- **مفاهیم**: رشته‌ها، مقایسه، حلقه‌ها
- **ورودی**: یک رشته از کاربر
- **خروجی**: اعلام پالیندروم بودن یا نبودن رشته

#### `reverse_string.cpp`
- **هدف**: برعکس کردن یک رشته
- **مفاهیم**: رشته‌ها، حلقه‌ها، اندیس‌گذاری
- **ورودی**: یک رشته از کاربر
- **خروجی**: رشته‌ی معکوس‌شده از آخر به اول

#### `search_string_in_text.cpp`
- **هدف**: جستجوی یک رشته داخل رشته‌ی دیگر و محاسبه تعداد تکرار آن
- **مفاهیم**: رشته‌ها، حلقه‌ها، جستجوی زیررشته
- **ورودی**: دو رشته که توسط کاربر وارد می‌شود
- **خروجی**: وجود یا عدم وجود رشته و تعداد تکرار آن
