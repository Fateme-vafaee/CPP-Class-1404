# 01-Basic - C++ Fundamentals

## English

### Overview
This folder contains basic C++ programming exercises covering fundamental concepts including loops, conditional statements, number operations, and pattern printing.

### Files Description

#### `average_of_numbers.cpp`
- **Purpose**: Calculates the average of a series of numbers
- **Concepts**: Input/output, loops, arithmetic operations
- **Input**: A collection of numbers from the user
- **Output**: The calculated average

#### `average_of_number_by_while.cpp`
- **Purpose**: Calculates the average of numbers using a while loop
- **Concepts**: While loop, counter variables, accumulation pattern
- **Input**: Numbers provided by user
- **Output**: Average value computed iteratively

#### `convert_second.cpp`
- **Purpose**: Converts seconds into hours, minutes, and seconds
- **Concepts**: Modulo operator, integer division, time conversion
- **Input**: Total seconds as an integer
- **Output**: Equivalent hours, minutes, and remaining seconds

#### `print_prime_numbers.cpp`
- **Purpose**: Identifies and prints prime numbers within a range
- **Concepts**: Nested loops, conditional logic, prime number algorithm
- **Input**: Range boundaries
- **Output**: All prime numbers in the specified range

#### `print_stars.cpp`
- **Purpose**: Creates patterns using asterisks (*) symbols
- **Concepts**: Nested loops, pattern generation
- **Input**: Pattern dimensions (rows/columns)
- **Output**: Visual star patterns



## فارسی

### نمای کلی
این پوشه شامل تمرین‌های برنامه‌نویسی پایه‌ای C++ است که مفاهیم بنیادی شامل حلقه‌ها، دستورات شرطی، عملیات اعداد و چاپ الگوها را پوشش می‌دهد.

### توضیح فایل‌ها

#### `average_of_numbers.cpp`
- **هدف**: محاسبه میانگین یک سری اعداد
- **مفاهیم**: ورودی/خروجی، حلقه‌ها، عملیات حسابی
- **ورودی**: مجموعه‌ای از اعداد از کاربر
- **خروجی**: میانگین محاسبه شده

#### `average_of_number_by_while.cpp`
- **هدف**: محاسبه میانگین اعداد با استفاده از حلقه while
- **مفاهیم**: حلقه while، متغیرهای شمارنده، الگوی انباشت
- **ورودی**: اعدادی که توسط کاربر فراهم شده‌اند
- **خروجی**: مقدار میانگین محاسبه شده به صورت تکراری

#### `convert_second.cpp`
- **هدف**: تبدیل ثانیه به ساعت، دقیقه و ثانیه
- **مفاهیم**: عملگر modulo، تقسیم صحیح، تبدیل زمان
- **ورودی**: کل ثانیه‌ها به عنوان عدد صحیح
- **خروجی**: ساعات، دقایق و ثانیه‌های باقی‌مانده

#### `print_prime_numbers.cpp`
- **هدف**: شناسایی و چاپ اعداد اول در یک محدوده
- **مفاهیم**: حلقه‌های تو در تو، منطق شرطی، الگوریتم اعداد اول
- **ورودی**: مرزهای محدوده
- **خروجی**: تمام اعداد اول در محدوده مشخص شده

#### `print_stars.cpp`
- **هدف**: ایجاد الگوهایی با استفاده از نمادهای ستاره (*)
- **مفاهیم**: حلقه‌های تو در تو، تولید الگو
- **ورودی**: ابعاد الگو (سطرها/ستون‌ها)
- **خروجی**: الگوهای بصری ستاره‌ها
